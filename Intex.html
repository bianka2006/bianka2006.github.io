Soporte Técnico - Segundo Trimestre

Unidad Central de Procesamiento (CPU): También conocida como procesador, es el cerebro de la computadora. Realiza operaciones aritméticas y lógicas, ejecuta instrucciones de programas y coordina las operaciones de otros componentes.

Placa madre (Motherboard): Es la plataforma principal que conecta y permite la comunicación entre todos los componentes de la computadora. Contiene el chipset, ranuras de expansión, conectores y puertos para periféricos.

Memoria RAM (Random Access Memory): Almacena temporalmente datos y programas en uso para que la CPU pueda acceder a ellos rápidamente. Es volátil, lo que significa que pierde los datos cuando la computadora se apaga.

Almacenamiento: Puede ser de dos tipos principales:

Disco Duro (HDD): Ofrece almacenamiento a largo plazo para el sistema operativo, programas y archivos. Unidad de Estado Sólido (SSD): Proporciona almacenamiento más rápido y confiable que los discos duros tradicionales. Tarjeta gráfica (GPU): Maneja el procesamiento de gráficos y mejora el rendimiento visual en juegos y aplicaciones intensivas en gráficos.

Fuente de alimentación (PSU): Suministra energía eléctrica a todos los componentes de la computadora. Se conecta a la toma de corriente y distribuye la electricidad de manera segura.

Unidades Ópticas: Aunque cada vez son menos comunes, algunas PCs todavía incluyen unidades de CD/DVD/Blu-ray para leer o grabar discos ópticos.

Tarjetas de expansión: Incluyen tarjetas de sonido, tarjetas de red y otras tarjetas que proporcionan funcionalidades adicionales a la placa madre.

Periféricos: Incluyen dispositivos de entrada y salida, como teclado, mouse, monitor, impresora, escáner, etc.

Carcasa (Case): Es la estructura que alberga todos los componentes de la PC y proporciona protección y refrigeración.

Sistema de enfriamiento: Incluye ventiladores y disipadores de calor para mantener los componentes, especialmente la CPU y la GPU, a temperaturas adecuadas.

Conectividad: Puertos y ranuras para la conexión de dispositivos externos, como USB, HDMI, Ethernet, entre otros.

Soporte Técnico - Tercer Trimestre



Pasos para crear una Red LAN:

Planificación:

Define el propósito de la red y los recursos que compartirán los dispositivos. Determina la topología de red (estrella, bus, anillo, malla, etc.). Identifica la cantidad de dispositivos que se conectarán a la red. Selecciona el Equipamiento:

Router: Proporciona la conexión a Internet y ayuda a dirigir el tráfico en la red. Switch o Hub: Facilita la comunicación entre dispositivos dentro de la red. Cables Ethernet: Utiliza cables CAT6 o superiores para conectar dispositivos al switch o router. Tarjetas de red: Asegúrate de que los dispositivos tengan tarjetas de red incorporadas o externas. Configura el Router:

Conecta el router a la fuente de Internet (módem). Accede al panel de configuración del router a través de un navegador web. Configura la conexión a Internet (DHCP, PPPoE, etc.). Asigna una dirección IP al router. Conecta Dispositivos:

Conecta el switch al router. Conecta los dispositivos (computadoras, impresoras, etc.) al switch usando cables Ethernet. Configuración de IP:

Puedes configurar las direcciones IP manualmente o permitir que el router asigne automáticamente direcciones IP a través de DHCP. Nombre de la Red (SSID):

Si estás utilizando una red inalámbrica, establece un nombre para la red (SSID) y configura la seguridad (WPA2 o WPA3). Seguridad de la Red:

Configura contraseñas seguras para el router y, si es necesario, para las conexiones inalámbricas. Considera la posibilidad de utilizar un cifrado fuerte para proteger la comunicación en la red. Pruebas de Conexión:

Verifica la conectividad entre los dispositivos. Asegúrate de que cada dispositivo puede acceder a la red y, si es necesario, a Internet. Consideraciones Adicionales:

Puedes configurar recursos compartidos en las computadoras, como carpetas o impresoras compartidas. Implementa políticas de seguridad, como firewalls, para proteger la red. Documentación:

Documenta la configuración de la red, incluyendo direcciones IP, nombres de dispositivos y contraseñas. Especificaciones de Hardware:

Procesador (CPU): Marca, modelo y velocidad de reloj.

Memoria RAM: Capacidad (en gigabytes), tipo y velocidad.

Almacenamiento: Capacidad del disco duro o SSD, tipo de almacenamiento (HDD o SSD), y velocidad.

Tarjeta Gráfica (GPU): Marca, modelo y memoria dedicada.

Placa Madre: Marca y modelo.

Fuente de Alimentación (PSU): Potencia en vatios y eficiencia.

Sistema de Enfriamiento: Tipo de refrigeración (aire o líquida) y marca/modelo si es un sistema de refrigeración adicional.

Caja/Chasis: Marca y modelo.

Complementos y Periféricos:

Monitor: Tamaño, resolución y características especiales (por ejemplo, frecuencia de actualización alta para gaming).

Teclado y Mouse: Marca, modelo y características especiales (retroiluminación, inalámbrico, etc.).

Altavoces o Auriculares: Marca, modelo y calidad de audio.

Webcam: Especificaciones si está incluida.

Unidad Óptica: Si la PC tiene una unidad de CD/DVD/Blu-ray.

Tarjeta de Red/Wi-Fi: Especifica si tiene conexión Ethernet integrada y/o Wi-Fi.

Software:

Sistema Operativo: Incluido y versión.

Software Adicional: Lista de software preinstalado si lo hay.

Conectividad:

Puertos: Enumera los tipos y cantidad de puertos USB, HDMI, DisplayPort, etc. Conectividad Inalámbrica: Especifica las capacidades Wi-Fi y Bluetooth.

Estado y Garantía:

Estado de la PC: Descripción del estado físico y funcional.

Garantía: Detalles sobre la garantía restante, si la hay.

Rendimiento y Uso:

Uso Anterior: Describe el uso anterior de la PC, como si ha sido utilizada para juegos, diseño gráfico, programación, etc.

Rendimiento: Puedes proporcionar información sobre el rendimiento en aplicaciones específicas o benchmarks si es relevante. Precio y Condiciones de Venta:

Precio: Establece el precio de venta. Condiciones de Pago y Entrega: Especifica si hay condiciones especiales para el pago y cómo se llevará a cabo la entrega.

Diseño Web y Desarrollo Web - Primer Trimestre

El primer trimestre del curso de Diseño Web y Desarrollo Web se enfoca en introducir a los estudiantes en los conceptos básicos del diseño y desarrollo de sitios web. Diseño Web y Desarrollo Web - Segundo Trimestre

Características y Conceptos Clave de Soluciones Web

Usabilidad y Experiencia del Usuario (UX)

La usabilidad es esencial para el éxito de cualquier solución web. La interfaz de usuario debe ser intuitiva, facilitando la navegación y la interacción. La experiencia del usuario (UX) se centra en la satisfacción general del usuario durante su visita al sitio.

Accesibilidad y Diseño Inclusivo

La accesibilidad garantiza que todas las personas, independientemente de sus habilidades o discapacidades, puedan acceder al contenido del sitio. El diseño inclusivo busca llegar a la mayor audiencia posible, creando experiencias web para todos.

Diseño Responsivo

El diseño responsivo asegura que un sitio web se vea y funcione bien en una variedad de dispositivos y tamaños de pantalla. Esto incluye computadoras de escritorio, tabletas y teléfonos móviles. Un diseño flexible mejora la experiencia del usuario y el posicionamiento en motores de búsqueda.

Optimización para Motores de Búsqueda (SEO)

La optimización para motores de búsqueda (SEO) es esencial para mejorar la visibilidad en los resultados de búsqueda. Estrategias como la selección de palabras clave, la creación de contenido de calidad y la optimización técnica contribuyen a un mejor ranking en los motores de búsqueda.

Seguridad y Privacidad

La seguridad web es crítica para proteger tanto el sitio como los datos de los usuarios. Esto incluye el uso de protocolos seguros (HTTPS), la gestión adecuada de contraseñas y la actualización constante de software para abordar vulnerabilidades.

Interactividad y Tecnologías Emergentes

La interactividad mejora la participación del usuario. El uso de tecnologías emergentes, como AJAX, permite cargar contenido de manera dinámica sin recargar la página completa. Asimismo, tecnologías como WebAssembly y APIs modernas brindan experiencias web más avanzadas.

Análisis de Datos y Métricas

La recopilación y análisis de datos son fundamentales para comprender el comportamiento de los usuarios. Herramientas como Google Analytics proporcionan información valiosa sobre el tráfico, la interacción y el rendimiento del sitio web.

Integración con Redes Sociales

La presencia en redes sociales es clave en la actualidad. La integración de botones para compartir y la exhibición de feeds sociales en un sitio web facilitan la conexión y la participación social.
